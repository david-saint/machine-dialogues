# Research Harness

A framework for running and viewing comparative experiments between Large Language Models.

## Project Structure

- `src/research_harness/`: Python backend for running experiments and generating transcripts.
- `viewer/`: React/Vite/TypeScript frontend for visualizing conversational transcripts and evaluations.
- `experiments/`: YAML definitions for various model-to-model probes and comparisons.
- `transcripts/`: Markdown conversation logs generated by the research harness.
- `plans/`: Design documents and project roadmaps.

## Getting Started

### Backend (Python)
The backend uses `uv` for dependency management.
```bash
# Install dependencies
uv sync

# Run the CLI
uv run rh --help
```

### Viewer (Frontend)
The viewer is built with React and Vite.
```bash
cd viewer
npm install
npm run dev
```
